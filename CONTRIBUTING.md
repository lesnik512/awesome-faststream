# Contributing

Contributions are welcome. Thank you for helping make this list better.

## Guidelines

- Make sure the entry is related to [FastStream](https://github.com/ag2ai/faststream) — a library, tool, template, broker/transport, integration, or learning resource for it.
- One entry per pull request, so it can be reviewed and discussed on its own.
- Add the entry to the most appropriate section, keeping entries alphabetically ordered within a section where possible.
- Use the format: `- [name](link) - Description.`
- The description must start with a capital letter and end with a period.
- Keep descriptions short, objective, and free of marketing language.
- Prefer a link to the project's source repository.
- New additions should be actively maintained and useful to FastStream users.

## How to contribute

1. Fork this repository.
2. Add your entry following the guidelines above.
3. Open a pull request with a short, clear title (for example, `Add modern-di-faststream`).

Entries that don't fit, are unmaintained, or are duplicates may be declined. Thanks for contributing!
